# Causal-Normative Networks

This repository contains the LaTeX source for Brett Reynolds's paper on causal-normative networks: typed explanatory structures in which causal uptake, normative transitions, recognition, and constitutive status relations are coupled without reducing one to another.

## Build

The paper requires XeLaTeX.

```sh
make
```

The build uses `references-standalone.bib` so the repository can compile without Brett's shared bibliography checkout. Local literature files are intentionally excluded from GitHub.
